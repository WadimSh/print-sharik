# print-sharik

Веб-приложения оформления заказов на коммерческую печать на латексных шарах.

---

### Структура каталога проекта

Структура каталога верхнего уровня (папка src) выглядит следующим образом:
- index.tsx - этот файл является точкой входа; 
- App.jsx - это основной файл, в котором устанавливается маршрутизация приложения между страницами;
- page - в этой папке отражает маршруты приложения. Страницы содержат элементы из папки components. Страницы не имеют логики и могут работать только с маршрутами;
- components - эта папка содержит глобальные компоненты, которые содержат логику и элементы из папки ui, которыми управляют через пропсы; 
- ui - в этой папке содержаться "тупые" повторно используемые элементы интерфейса, отвечающие за отрисовку и осуществляющие действия с функциями передоваемыми через пропсы. Стили этих элементов вложены в js;
- context - в папке хранятся все файлы контекста React, которые используются на страницах приложения;
- utils - это место где размещаются служебные функции, которые неоднократно используются в проекте;
- assets - в этой папке храниться практически все, что не связано с кодом, все изображения, css-файлы, файлы шрифтов и т.д.
