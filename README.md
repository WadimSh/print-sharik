# print-sharik

Веб-приложения оформления заказов на коммерческую печать на латексных шарах.

---

### Структура проекта
Код разработке разделен на 5 слоев, а именно самый верхний слой - page, где лежат страницы приложения. Второй - components, основные блоки из которых строятся страницы. Слой - ui, в нем расположены элементы интерфейса из которых строятся блоки. Слои assets и utils - содержат все остальное.
